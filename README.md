## AI-playlist-generator

This playlist generator is powered by OpenAI's ChatGPT. It allows user to create a playlist of songs based on a given statement. The playlist will automatically be created into the user's Spotify account as a public playlist. 

App can be accessed at: https://ai-playlist-generator-1y3v.onrender.com/
